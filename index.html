Hello! My name is Ryan Folmer, and I am a first-year computer science student at Dalhousie University in Halifax, Nova Scotia. This is a website built for a web development intro course, but it also shows some projects I have worked on and will work on, as well as being a little personal place on the internet. I am interested in game development, cybersecurity, and robotics.

This website was built for CSCI 1170: Intro to Web Development. It should feature three pages, a navigation system, pictures, videos, audio, and other features used to show proficiency with HTML. For example, this paragraph and image are actually set up as a table.

I plan to work on a few things in the future, mainly a character mod for the game Your Only Move Is Hustle. I know modding is a good on-ramp to game development, so hopefully I get some good experience from that. I also want to finish a card game I made during high school. Currently, that game exists as a paper prototype, but in the future a digital version would be a good idea.
